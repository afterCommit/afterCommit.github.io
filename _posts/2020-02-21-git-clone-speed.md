1. 查找域名对应的ip地址
`nslookup github.global.ssl.fastly.Net`
`nslookup github.com`
2. 修改hosts文件
`vi etc/hosts`
3. 重启或更新
`sudo /etc/init.d/networking restart`
`sudo dscacheutil -flushcache`
